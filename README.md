# create_dump_db

Унифицированный скрипт для созданияи дампов базданных `Postgres` для инстаносов поднятых через docker compose
- Использует для работы файл `.env.dump`
- Обязательные переменные для работы указаны в `.env.example`

### Приемущества
- Интегрирутеся в уже работующий проект который запущен с использованнием docker-compose
